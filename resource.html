资源分享

HTML5

HTML5是HTML最新的修订版本，2014年10月由万维网联盟（W3C）完成标准制定。 HTML5的设计目的是为了在移动设备上支持多媒体。

CSS3

CSS 用于控制网页的样式和布局。 CSS3 是最新的 CSS 标准。 CSS3是CSS技术的升级版本，CSS3语言开发是朝着模块化发展的.

PHP/JSP

PHP 是一种创建动态交互性站点的强有力的服务器端脚本语言。 PHP 是免费的.对于像微软ASP这样的竞争者来说,PHP无疑是另一种高效率的选项。

WEB APP

WebApp是指基于Web的系统和应用，其作用是向广大的最终用户发布一组复杂的内容和功能。响应式网页设计是可用在WebApp开发中的。

JAVA

Java 是由Sun公司于1995年5月推出的高级程序设计语言。 Java可运行于多个平台.本教程通过简单的实例将让大家更好的了解JAVA编程语言。

C/C++

C++ 是一种中级语言，它是于 1979 年在贝尔实验室开始设计开发的。C++ 进一步扩充和完善了 C 语言，是一种面向对象的程序设计语言。
